1. What is the difference between the following 2 statements?
setTimeout(booyah, 2000);
setTimeout(booyah(), 2000);

Answer:
setTimeout(booyah, 2000); -> Call booyah after 2 second.
setTimeout(booyah(), 2000) -> booyah is going to be called Immediately.

4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Anwer:

The word relates with writing javascript as a completely separate file which means no JS code inside HTML even no function call inside html tag.
So for the sole purpose of separation of context we put files in the following format:

  • content (html)
  • presentation (css)
  • behavior (js)
