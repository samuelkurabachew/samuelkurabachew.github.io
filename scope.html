2. What is the difference between a method and function?

Answer: A function is called by name and works as a first-class variable while a method is property of an object that contains a function definition and only accessible from its object.

3. What does 'this' refer to when used in a Java method?

Answer: ‘this’ keyword in java refers to the current object.

4. What does 'this' refer to when used in a JavaScript method?

Answer: ‘this’ keyword in JavaScript refers to the object it belongs to. It has different values depending on where it is used: In a method, this refers to the owner object. Alone, this refers to the global object. In a function, this refers to the global object.

5. What does 'this' refer to when used in a JavaScript constructor function?

Answer: "this" refers to object created inside constructor.

7. What is a free variable in JavaScript?

Answer: Free variables are simply the variables that are neither locally declared nor passed as parameter but called by the current function

9. Write Javascript code for creating three Employee objects using the "new" keyword and a constructor function. Employee objects have the following fields: name, salary, position.

Answer: 

            function Employee (name, salary, position){
                this.name = name;
                this.salary = salary;
                this.position = position;
            }

            var emp1 = new Employee("Samuel", 150000, "Full Stack Developer");
            var emp2 = new Employee("Sam", 200000, "Data Engineer");
            var emp3 = new Employee("Ted", 250000, "Data Scientist");

10. Write a Javascript function that takes any number of input arguments and returns the product of the arguments.

Answer: 

            function product(...nums){
                var result = 1;
                for(var i=0; i < nums.length; i++) { 
                    result *=nums[i]; 
                } 
                return result; 
            }
