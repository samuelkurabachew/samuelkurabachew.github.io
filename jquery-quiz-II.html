1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent. 

            //code fragment 1
            $("li").each(function(idx, e) {
            $(e).css(“color”, “yellow”); });
            //code fragment 2
            $("li").each(function() {
            $(this).css(“color”, “yellow”); });
            //code fragment 3
            $("li").each(function(idx) {
            $(this).css(“color”, “yellow”); });

Answer:
Interms of output all code fragment yield the same result, turning the li content color to yellow. The difference is that the first fragment uses index and DOM element to apply the color where in the second and third we use 'this' keyword to access the jquery object instead of regular DOM element.
